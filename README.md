# Message Board

### NodeJS API

## Installation

1. Unzip into folder
2. `npm install`
3. `npm fund`
4. `npm start`

### Server running at:

`http://localhost:3000`

## Routes

- **GET** [/node/users](http://localhost:3000/node/users) Get all users
- **POST** [/node/users](http://localhost:3000/node/users) Add a user
- **DELETE** [/node/users/id](http://localhost:3000/node/users/1) Remove a user by id
