# Message Board

### NodeJS API

## Installation

1. Unzip into folder
2. `npm install`
3. `npm fund`
4. `npm start`

### Server running at:

`http://localhost:3000`

## Routes

- `GET /node/users` Get all users
- `POST /node/users` Add a user
- `DELETE /node/users/id` Remove a user by id
